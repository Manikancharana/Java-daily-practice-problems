 ArrayList is :  ArrayList internally uses a dynamic array. When the array becomes full, 
                 it creates a new array with 1.5x capacity and copies old elements. It provides O(1) access using index 
                 but insertion/deletion in middle takes O(n) due to shifting.
   Declaration : ArrayList<Integer> arraylist = new ArrayList<>();


   Differecnce b/t CustomArrayList and ArrayList : "A custom ArrayList is manually implemented using arrays and resizing logic, mainly for learning purposes. 
   Java’s ArrayList is a fully optimized, generic, dynamic array implementation that provides many built-in methods and handles resizing internally."
